# Stock Revenue Analysis

Analyzes and visualizes stock price vs revenue for Tesla and GameStop.

## What it does
- Fetches historical stock data using yfinance API
- Scrapes quarterly revenue data using BeautifulSoup
- Plots interactive dual-axis charts (stock price + revenue) using Plotly

## Tools & Libraries
Python | yfinance | BeautifulSoup | Pandas | Plotly

## How to run
```bash
pip install yfinance pandas requests beautifulsoup4 plotly
```
Then run all cells in `final_project_stock_revenue.ipynb`

## Charts
- Tesla Stock Price vs Quarterly Revenue (2010–2025)
- GameStop Stock Price vs Quarterly Revenue (2002–2025)
